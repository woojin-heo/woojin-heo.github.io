#### select문이란?
데이터베이스에서 정보를 검색하는 DQL 유형의 명령어 (SQL문장의 종류-링크삽입)


#### select문의 기능(검색, 조회) 3가지
1. projection
	- query에 반환되는 테이블의 열을 선택한다.
	- 컬럼선택, 열 단위 추출
```
select employee_id
from employees;
```

2. selection
	- query에 의해 반환되는 테이블의 행을 선택한다.
```
select *
from employees
where employoee_id = 100;
```

3. join
	- 서로 다른 테이블에 저장된 데이터를 함께 가져온다.
